# @oada/lib-config

Shared configuration wrapper for OADA services and libraries.

Use this package to read normalized runtime configuration derived from
`oada.config.mjs` and environment variables.

## Scripts

Run from `oada/`:

- `yarn workspace @oada/lib-config run build`

## Notes

- `test` is currently a placeholder script in this package.
- This package is intended to centralize config loading behavior.
